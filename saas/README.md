## Azure SaaSAPI SDK for JavaScript

This package contains an isomorphic SDK for SaaSAPI.

### Currently supported environments

- Node.js version 8.x.x or higher
- Browser JavaScript

### How to Install

```bash
npm install microsoft.marketplace.saas
```
